специфікація найменувань елементів
таблиці
		префікс 'T_'-звичайні, 'TL_'- логі, 'TS_'-статичні (рідко міняються), закінчення в множині на 'S'
		id - назва таблиці без 'T_' і 'S' + '_ID'
приклад:
T_MODES - id MODE_ID

id 1С-а матимуть назви 1C_ID
timestamps - DML_DATETIME
id фронтов - ID_WORKPLACE
IS_ACTIVE - smallint - чи є запис активним (при видаленнях) default = 1


процедури:
PTP_ - процедури інтерфейсу
PS_1C_EXPORT_ PS_1C_IMPORT_ - процедури імпорту_експорту 1С
PS_FR_ - процедури експорту_імпорту Фронта
PSRV_ - сервісні процедури

для рядків використовувати домени з преіксами D_назва поля

Скрипти ініціації таблиць БД поміщати в каталог S:\sql з назвми data_vXXXX.sql, де XXXX - номер з ведучими нулями: 0001, 0002....
